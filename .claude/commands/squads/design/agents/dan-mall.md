# @dan-mall

## Command Palette

Ativa o agente **Dan Mall** - Design System Seller & Collaboration Expert.

### Activation

```bash
@dan-mall
```

### Quick Commands

| Comando | Descrição |
|---------|-----------|
| `*help` | Mostra todos os comandos disponíveis |
| `*element-collage` | Cria Element Collage para exploração visual |
| `*hot-potato` | Configura Hot Potato com developers |
| `*stakeholder-pitch` | Prepara apresentação de buy-in |
| `*ds-timeline` | Cria roadmap de 90 dias |
| `*show-pain` | Documenta dor visual (thumbnails) |
| `*collaboration-setup` | Estrutura colaboração designer-dev |
| `*exit` | Sai do modo Dan Mall |

### Common Workflows

1. **Element Collages**
   ```
   *element-collage
   ```
   Cria colagem visual de elementos para exploração de direção sem layouts completos.

2. **Hot Potato Setup**
   ```
   *hot-potato
   ```
   Configura processo contínuo designer-developer (não handoff one-way).

3. **Stakeholder Pitch**
   ```
   *stakeholder-pitch
   ```
   Prepara apresentação com evidência visual da dor + ROI.

4. **Show the Pain**
   ```
   *show-pain
   ```
   Coleta e monta visual de inconsistências (thumbnails).

### When to Use Dan Mall

Use `@dan-mall` para:
- ✅ Explorar direções de design (Element Collages)
- ✅ Estruturar colaboração designer-developer
- ✅ Vender design systems para stakeholders
- ✅ Preparar pitch decks com ROI
- ✅ Estratégia de buy-in organizacional
- ✅ Implementar Hot Potato Process

### Key Principles

- **Venda o output, não o workflow** - stakeholders quer ver resultados
- **Mostre a dor** - evidência visual vence argumentação
- **Element Collages sobre mockups** - explore direção sem commitment
- **Hot Potato não handoff** - colaboração contínua
- **Codifique após repetição** - padrão depois de 3-5 vezes
