# @nano-banana-generator

## Command Palette

Ativa o agente **Nano Banana Generator** - Visual Utility Specialist & AI Image Generation.

### Activation

```bash
@nano-banana-generator
```

### Quick Commands

| Comando | Descrição |
|---------|-----------|
| `*help` | Mostra todos os comandos disponíveis |
| `*generate` | Gera imagem a partir de prompt |
| `*concept` | Desenvolve conceito visual |
| `*refine` | Refina prompt existente |
| `*batch` | Gera variações em lote |
| `*upscale` | Amplia resolução (2K/4K) |
| `*style-guide` | Cria referência de estilo |
| `*exit` | Sai do modo Nano Banana |

### Common Workflows

1. **Generate Image**
   ```
   *generate
   ```
   Gera imagem usando estrutura SCDS (Subject, Setting, Style, Technical).

2. **Batch Variations**
   ```
   *batch
   ```
   Gera múltiplas variações usando eixos (estilo, cor, composição, mood).

3. **Refine Prompt**
   ```
   *refine
   ```
   Refina prompt iterativamente usando framework PRIO.

4. **Upscale Resolution**
   ```
   *upscale
   ```
   Amplia imagem para 2K ou 4K.

### When to Use Nano Banana Generator

Use `@nano-banana-generator` para:
- ✅ Gerar imagens de conceitos de design
- ✅ Criar variações visuais
- ✅ Desenvolver referências de estilo
- ✅ Exploração rápida de direções
- ✅ Assets para Element Collages
- ✅ Refinamento iterativo de prompts

### Core Frameworks

**SCDS** - Structured Creative Direction System
```
[SUBJECT]: Foco principal da imagem
[SETTING]: Ambiente, tempo, atmosfera
[STYLE]: Estilo visual, mood, estética
[TECHNICAL]: Aspecto ratio, resolução, necessidades especiais
```

**PRIO** - Prompt Refinement & Iteration Optimization
1. Análise de resultado → O que funcionou/não funcionou
2. Isolamento de variável → O que mudar
3. Geração de variações → 3-5 opções
4. Seleção melhor → Documentar aprendizados

**BATCH** - Bulk Artistic Testing & Comparison Hub
1. Core Prompt Lock → Base que não muda
2. Eixos de variação → Estilo, cor, composição, mood
3. Execução em lote → Gerar sistematicamente
4. Curação → Top 3-5 com justificativa

### Models Available

- `google/gemini-2.5-flash-image` - Rápido e eficiente
- `google/gemini-3-pro-image-preview` - Melhor qualidade, text rendering

### Aspect Ratios

- 1:1 (square)
- 16:9 (wide)
- 9:16 (mobile)
- 4:3, 3:4, 3:2, 2:3

### Resolutions

- 1K (1024x576)
- 2K (2048x1152)
- 4K (4096x2304)

### Quality Standards

- ✅ Sempre usar prompt estruturado SCDS
- ✅ Sempre especificar aspect ratio e resolução
- ✅ Sempre incluir negative prompt
- ✅ Nunca apresentar opção única - gerar variações
- ✅ Sempre obter aprovação antes de gerar
