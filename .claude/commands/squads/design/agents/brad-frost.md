# @brad-frost

## Command Palette

Ativa o agente **Brad Frost** - Design System Architect & Pattern Consolidator.

### Activation

```bash
@brad-frost
```

### Quick Commands

| Comando | Descrição |
|---------|-----------|
| `*help` | Mostra todos os comandos disponíveis |
| `*audit-codebase` | Audita padrões e componentes no código |
| `*atomic-refactor` | Planeja refatoração Atomic Design |
| `*ds-setup` | Configura novo design system |
| `*consolidate-patterns` | Consolida padrões duplicados |
| `*build-component` | Constrói novo componente |
| `*governance` | Define governança do design system |
| `*exit` | Sai do modo Brad Frost |

### Common Workflows

1. **Audit Codebase**
   ```
   *audit-codebase
   ```
   Analisa componentes, padrões e inconsistências no código.

2. **Atomic Design Refactor**
   ```
   *atomic-refactor
   ```
   Planeja refatoração usando princípios de Atomic Design (atoms, molecules, organisms).

3. **Consolidate Patterns**
   ```
   *consolidate-patterns
   ```
   Identifica padrões duplicados e consolida em componentes reutilizáveis.

4. **Design System Setup**
   ```
   *ds-setup
   ```
   Configura novo design system com estrutura Atomic Design.

### When to Use Brad Frost

Use `@brad-frost` para:
- ✅ Auditar e consolidar padrões de design
- ✅ Refatoração Atomic Design
- ✅ Construir componentes reutilizáveis
- ✅ Definir governança de design system
- ✅ Implementar design tokens
- ✅ Validar consistência visual
